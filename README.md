# Personal Portfolio Landing Page

 Project Overview

This is a simple and responsive Personal Portfolio Landing Page created using HTML and CSS only. The project was built to practice front-end development concepts such as page structure, Flexbox layout, responsive design, and clean styling.

 Features

 Responsive Navigation Bar
 Header with Personal Introduction
 About Me Section
 Skills Section
 Projects Section
 Contact Form
 Mobile-Friendly Responsive Layout
 Clean UI using CSS Flexbox

 Technologies Used

 HTML5
 CSS3
 Flexbox
